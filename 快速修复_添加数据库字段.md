# 快速修复：添加数据库字段

## 错误信息
```
"column t.unfinished_works does not exist"
```

## 解决方案

你需要在数据库中添加 `unfinished_works` 和 `finished_works` 字段。

### 方法 1：在 Vercel Dashboard 执行（推荐）

1. **打开 Vercel Dashboard**
   - 进入你的项目
   - 点击 Storage → AIGC-Studio_Database

2. **点击 "Open in Neon"** 按钮
   - 这会打开 Neon 数据库管理界面

3. **在 SQL Editor 中执行以下 SQL：**

```sql
ALTER TABLE teams 
ADD COLUMN IF NOT EXISTS unfinished_works JSONB DEFAULT '[]',
ADD COLUMN IF NOT EXISTS finished_works JSONB DEFAULT '[]';
```

4. **点击 "Run" 执行**

### 方法 2：使用 psql 命令行（如果你有数据库连接信息）

如果你有数据库连接字符串，可以使用命令行：

```bash
psql "你的DATABASE_URL" -c "ALTER TABLE teams ADD COLUMN IF NOT EXISTS unfinished_works JSONB DEFAULT '[]', ADD COLUMN IF NOT EXISTS finished_works JSONB DEFAULT '[]';"
```

### 方法 3：在 Vercel Postgres SQL Editor 执行

如果你使用的是 Vercel Postgres（不是 Neon）：

1. Vercel Dashboard → 你的项目 → Storage → Postgres
2. 点击 "SQL Editor"
3. 粘贴上面的 SQL
4. 点击执行

## 验证

执行成功后，刷新你的网站页面，应该就能正常工作了！

测试 API：
- 访问 `https://你的域名/api/teams`
- 应该返回 JSON 数据，不再报错

## 为什么需要这个？

这个字段是用于存储"未完成作品"和"已完成作品"的图片 URL 列表。如果不添加这些字段，当代码尝试保存作品数据时就会报错。
