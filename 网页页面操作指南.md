# 页面操作指南（中控台首页 / App 页面）

本文从**客户/使用者视角**说明如何操作网站（不包含部署、数据库配置等开发内容）。  
适用页面：**“剧变时代中控台”**（对应 `src/App.tsx`）。

---

## 0. 新手 5 分钟上手（必读）

目标：第一次打开就能 **看懂数据 → 快速定位 → 会解锁更新 → 会保存/备份 → 会发日报**。

### 0.1 先判断你是不是“多人共享”

看页面右上角（桌面端可见）：

- 🟢 **云端数据库**：多人共享、换设备也同步
- 🟡 **本地模式**：只保存在当前浏览器（清缓存会丢）

### 0.2 30 秒看全局

- **Total Progress**：全局平均进度
- **Active Staff**：成员总数
- **资金消耗**：`¥实耗 / ¥预算`（超预算会变红＝风险）

### 0.3 60 秒学会定位

在搜索框 **“搜索成员、任务...”** 输入：

- 输入**组名**：过滤出对应小组
- 输入**成员名**：过滤出包含该成员的小组

清空输入框即可恢复全部。

### 0.4 60 秒学会看任务（不需要密码）

- **看小组任务**：在组卡片找到 **Pending Tasks** → 鼠标悬停（hover）展开“全部任务”
- **看某个人任务**：鼠标悬停到成员卡片上，会弹出该成员任务浮窗

### 0.5 30 秒学会解锁更新（需要密码）

每个组卡片右上角都有：**UPDATE / 更新（锁按钮）**  
点击后输入：

- **组密码**：只解锁本组（推荐普通负责人）
- **管理员密码 `8888`**：解锁所有组 + 出现管理员工具栏

### 0.6 更新后必做：保存 + 备份（防丢核心）

管理员模式（`8888`）才会看到顶部按钮：

- **保存设置**：把修改保存（云端模式也会写入云端；失败也会至少保存到本地）
- **备份数据**：导出 `.json`（建议按日期存档）

### 0.7 一键发日报（管理员）

- 点 **生成日报** → 提示“已复制到剪贴板” → 飞书/微信群直接粘贴发送

## 1. 页面结构速览（你会看到什么）

- **顶部通告条**：有公告时会在页面最上方展示（📣 图标）。
- **顶部导航区**：
  - **Logo 与“剧变时代”标识**
  - **使用指南**：点击“使用指南”会直接跳转到飞书文档（外部打开）。
  - **主题切换**：桌面端可切换 4 种主题（黑色 / 深蓝 / 淡蓝 / 绿色）。
  - **数据状态指示器**（桌面端可见）：🟢云端数据库 / 🟡本地模式
  - **资金消耗**：全局实耗/预算汇总（超预算会变红）
- **页面概览区**：Total Progress / Active Staff
- **快捷入口**：剧查查榜单（`juchacha.html`）/ 作品展示（`works.html`）

---

## 2. 搜索（最快定位人/组）

在页面中部搜索框：**“搜索成员、任务...”**  
支持：

- 搜索**小组名称**
- 搜索**成员姓名**

> 提示：搜索结果会直接过滤下方小组卡片列表。

---

## 3. 小组卡片（如何读懂“进度/预算/风险”）

每张卡片通常包含：

- **组名 + 状态标签**（normal / urgent / review / done）
- **SIZE**（成员数量）与 **PROGRESS**（进度百分比）
- **每日交付**
- **预算（预期）**与**实耗**（超预算会标红）
- **进度条**
- **右上角 UPDATE/更新（锁）按钮**：用于解锁编辑

---

## 4. 任务查看（无需权限也能看）

### 4.1 小组任务（总目标）

在组卡片里找到 **Pending Tasks**：

- 默认只展示前几条
- **鼠标悬停（hover）**可展开“全部任务”浮窗

### 4.2 成员任务

“全部任务”浮窗里下半部分是 **成员任务**：

- 显示每个成员的待办数量（PENDING）
- 可查看每条任务的完成/未完成状态

### 4.3 快速查看某个成员的任务（悬停）

在成员卡片上悬停，会出现该成员的任务浮窗。

---

## 5. 封面图 / 参考图（Key Visual）

每个小组右侧通常有：

- **封面图（Key Visual）**：9:16 预览
- **参考图缩略图**

> 未解锁一般只能查看；上传/设置需要解锁（见第 7 章）。

---

## 6. 账号支出（查看每组充值/扣费记录）

每个小组右侧有 **账号支出** 模块：

- 展示平台（即梦/海螺/Vidu）、金额、时间、备注
- 便于审计“这组在哪个平台花了多少钱”

> 新增/删除支出记录通常需要解锁（见第 7/8 章）。

---

## 7. 解锁更新（最关键）

### 7.1 解锁入口

每个小组卡片右上角按钮：**UPDATE / 更新（锁按钮）**

### 7.2 输入哪种密码

点击后会提示：**“请输入【组名】的密码”**：

- **组密码**：只解锁该组
- **管理员密码 `8888`**：解锁所有组 + 进入管理员模式

### 7.3 解锁成功后你能做什么（常见）

- 新增/删除小组任务、成员任务，标记完成/取消完成
- 添加/编辑/删除成员（可能需要进入删除模式）
- 编辑小组信息（任务描述、工作量、预算、实耗、进度、状态、备注等）
- 上传/设置封面图与参考图
- 新增/删除支出记录

> 只看数据不需要解锁；只有要“改数据”才需要密码。

---

## 8. 具体操作步骤（照做即可）

### 8.1 标记任务完成 / 取消完成

1. 打开任务浮窗（Pending Tasks 悬停）
2. 点击任务左侧的方框图标切换完成状态

### 8.2 新增小组任务（总目标）

1. 打开“全部任务”浮窗
2. 在“新增小组任务…（回车添加）”输入内容
3. 按 Enter 或点“+”添加

### 8.3 删除小组任务

1. 打开“全部任务”浮窗
2. 找到任务右侧垃圾桶图标 → 删除（仅解锁后可见）

### 8.4 新增成员任务

1. 打开“全部任务”浮窗
2. 在“成员任务”找到目标成员
3. 在“新增成员任务…（回车添加）”输入内容
4. 按 Enter 或点“+”添加

### 8.5 删除成员任务

在成员任务条目右侧点击垃圾桶（仅解锁后可见）。

### 8.6 添加成员 / 删除成员（删除模式）

在组卡片的成员区（Execution Team）：

- **添加成员**：点 `ADD` → 按弹窗填写并确认
- **删除成员**：点 `DEL` 进入删除模式 → 成员卡上会出现删除按钮 → 删除完成后再点 `DEL` 退出

### 8.7 上传封面图 / 参考图

1. 鼠标移到封面图区域
2. 点击出现的“设置/上传”入口
3. 选择图片上传并等待完成

### 8.8 添加/删除支出记录

- **添加**：在“账号支出”区域点“+” → 选择平台/金额 → 可写备注 → 保存
- **删除**：支出条目右侧垃圾桶 → 删除

---

## 9. 管理员模式（仅 `8888` 可见）

管理员模式会在顶部显示“管理员模式”标识，并出现工具栏按钮：

- **生成日报**：一键复制日报到剪贴板（可直接群里粘贴）
- **保存设置**：保存全部修改（本地备份 + 云端模式写云端）
- **备份数据**：导出 JSON
- **恢复数据**：导入 JSON
- **重置**：恢复初始数据（谨慎）

并出现 **发布通告** 输入框：输入后会显示在页面顶部通告条。

---

## 10. 新手最常踩的坑（请按这条避开）

- **改了但没保存**：刷新后没变化 → 记住改完就点“保存设置”
- **把本地模式当云端用**：换设备数据不见 → 先看是否 🟢 云端数据库
- **不备份 JSON**：误操作无法回滚 → 重要改动后立刻“备份数据”
- **到处用管理员密码**：误删风险更高 → 普通情况用组密码只解锁一组
- **不知道 hover 能看任务**：Pending Tasks / 成员卡片都支持悬停快速查看
