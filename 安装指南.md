# Node.js 安装指南

## 问题
系统提示 `'npm' 不是内部或外部命令`，说明 Node.js 和 npm 尚未安装。

## 解决方案

### 方法一：从官网下载安装（推荐）

1. **访问 Node.js 官网**
   - 打开浏览器，访问：https://nodejs.org/
   - 或直接访问中文站：https://nodejs.org/zh-cn/

2. **下载 LTS 版本（推荐）**
   - 点击下载 "LTS"（长期支持版本）
   - 当前推荐版本：v20.x 或 v18.x
   - 下载 Windows Installer (.msi) 文件

3. **安装 Node.js**
   - 双击下载的 .msi 文件
   - 按照安装向导操作：
     - 点击 "Next"
     - 接受许可协议
     - 选择安装路径（默认即可）
     - **重要**：确保勾选 "Add to PATH" 选项
     - 点击 "Install"
   - 等待安装完成

4. **验证安装**
   - 关闭当前命令行窗口
   - 重新打开 PowerShell 或 CMD
   - 运行以下命令验证：
   ```bash
   node --version
   npm --version
   ```
   - 如果显示版本号，说明安装成功

### 方法二：使用包管理器安装

#### 使用 Chocolatey（如果已安装）

```powershell
choco install nodejs
```

#### 使用 Winget（Windows 10/11）

```powershell
winget install OpenJS.NodeJS.LTS
```

### 方法三：使用 NVM（Node Version Manager）

如果你需要管理多个 Node.js 版本：

1. 下载 NVM for Windows：
   - 访问：https://github.com/coreybutler/nvm-windows/releases
   - 下载 `nvm-setup.exe`

2. 安装后运行：
   ```bash
   nvm install lts
   nvm use lts
   ```

## 安装后操作

1. **重新打开终端**
   - 关闭当前所有命令行窗口
   - 重新打开 PowerShell 或 CMD

2. **验证安装**
   ```bash
   node --version
   npm --version
   ```

3. **进入项目目录并安装依赖**
   ```bash
   cd "C:\Users\Administrator\Desktop\AIGC Project\React+TypeScript版本"
   npm install
   ```

4. **启动开发服务器**
   ```bash
   npm run dev
   ```

## 常见问题

### 问题1：安装后仍然提示找不到命令

**解决方案：**
1. 检查 Node.js 是否真的安装了：
   - 打开文件资源管理器
   - 导航到：`C:\Program Files\nodejs\`
   - 查看是否有 `node.exe` 和 `npm.cmd` 文件

2. 手动添加到 PATH：
   - 右键"此电脑" → "属性" → "高级系统设置"
   - 点击"环境变量"
   - 在"系统变量"中找到 `Path`
   - 点击"编辑" → "新建"
   - 添加：`C:\Program Files\nodejs\`
   - 点击"确定"保存
   - **重新打开命令行窗口**

### 问题2：npm 命令很慢

**解决方案：**
使用国内镜像源（推荐使用淘宝镜像）：

```bash
npm config set registry https://registry.npmmirror.com
```

验证：
```bash
npm config get registry
```

### 问题3：权限问题

如果遇到权限错误，可以：
1. 以管理员身份运行 PowerShell
2. 或使用 `--global` 参数时添加 `--force` 参数
